¡Bienvenido a mi repositorio! Este espacio está dedicado a documentar soluciones tecnológicas creados para resolver cuellos de botella operativos, reducir la carga de tareas manuales y optimizar flujos de trabajo.

## 🚀 Proyectos Destacados

### 1. Extractor Inteligente de Cotas CAD (Python + IA)
Una aplicación web interactiva diseñada para procesar planos técnicos exportados de AutoCAD y consolidar sus datos automáticamente.

* **El Problema:** La extracción manual de medidas y nomenclaturas desde imágenes de planos es un proceso lento y propenso a errores humanos.
* **La Solución:** Un script que recibe imágenes masivas, aplica procesamiento digital para engrosar trazos técnicos y utiliza inteligencia artificial para leer los textos. Finalmente, limpia, ordena y exporta los datos a una cuadrícula estructurada.
* **Tecnologías:** `Python`, `Streamlit`, `Google Cloud Vision API`, `OpenCV`, `Pandas`, `Regex`.
* **Nota de uso:** Para ejecutar este script en local, es necesario contar con un archivo `credentials.json` válido de Google Cloud (no incluido en este repositorio por seguridad).

### 2. Orquestador Automático de Archivos (Python)
Script diseñado para la gestión y limpieza de directorios locales o en la nube.

* **El Problema:** La acumulación desordenada de reportes, facturas y documentos genera pérdida de tiempo en la búsqueda de información.
* **La Solución:** Un sistema que escanea carpetas masivas, identifica las extensiones de los archivos y los reubica automáticamente en directorios categorizados según reglas predefinidas.
* **Tecnologías:** `Python` (Librerías `os`, `shutil`).

### 3. Motor de Mensajería y Notificaciones (Google Apps Script)
Solución de comunicación automatizada integrada directamente en el ecosistema de Google Workspace.

* **El Problema:** El envío repetitivo de notificaciones o correos estandarizados consume tiempo valioso de gestión.
* **La Solución:** Un script alojado en la nube que lee bases de datos en Google Sheets y dispara mensajes personalizados automáticamente basándose en condiciones lógicas.
* **Tecnologías:** `Google Apps Script`, `JavaScript`, `Google Sheets API`.
